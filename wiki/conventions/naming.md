# Naming Conventions

This document outlines the naming conventions used throughout the codebase to ensure consistency and readability.

## Files

Filenames use **kebab-case**.

Examples:
- `active-users.js`
- `NorthAmericaRevenueChart.jsx`

## Components

React component filenames and their main export use **PascalCase**.

Examples:
- `ActiveUsersCard.jsx`
- `NorthAmericaRevenueChart.jsx`

## Variables and Functions

Variables and functions generally follow **camelCase**.

## Constants

Constants are typically declared using **UPPER_SNAKE_CASE**.

Examples:
- `DEFAULT_API_TIMEOUT`
- `MAX_RETRIES`
